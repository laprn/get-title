URL to Title

使い方

urlからhtmlのタイトル要素の中身を取得します。http(s)://から始まるURLを入力し、送信を押すとタイトルが取得できます。

curlにも対応しています。以下のように入力するとタイトルが取得できます。

curl https://cocno-title-fetcher.herokuapp.com/<url>

例：curl https://cocno-title-fetcher.herokuapp.com/https://sample.com

戻り値

戻り値はjsonで、以下のようになります。

message 結果のメッセージ
query 送信したクエリー
status ステータス
title タイトル

title:

{% with messages = get_flashed_messages() %} {% if messages %}
    {% for message in messages %}
  • {{ message['title'] }}
    • {% endfor %}
{% endif %} {% endwith %}
